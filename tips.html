SingleCellMQC

8 Tips

Compiled: 2025-04-26 Written by Jiaying Zeng

Here are some helpful tips for you to use SingleCellMQC more smoothly.

8.1 Global memory setting

When you initiate the work, the default global memory limit is 500MB, which is often insufficient, so it’s recommended to set it to 1GB or higher. For example, you can set global memory limit to 1GB (1024 MB * 1024 KB/MB * 1024 B/KB).

library(future)
options(future.globals.maxSize = 1024 * 1024 * 1024)

8.2 Report progress updates

Turning on the global progress update report feature before your analysis is suggested, so that you could keep better track of the whole QC progress.

library(progressr)
handlers(global = TRUE)
